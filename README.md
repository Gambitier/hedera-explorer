## Description

This repo provides wrapper API for [Hedera's mirror node REST API](https://docs.hedera.com/hedera/sdks-and-apis/rest-api)

Status

- [x] Query account balances

- [x] Query list of transactions

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
